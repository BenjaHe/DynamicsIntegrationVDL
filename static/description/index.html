Intégration du webshop avec Microsoft Dynamics

Architecture de l'intégration

Microsoft Dynamics n'accepte que les fichiers CSV et a besoin de champs spécifiques qui n'existe pas dans Odoo (ref ou id spécifiques).

Le fonctionnement choisi est donc d'ajouter les champs nécessaire à Dynamics et d'exporter les données à l'aide d'un job Talend qui ira lire directement dans la DB d'Odoo pour créer les tables nécessaires.
